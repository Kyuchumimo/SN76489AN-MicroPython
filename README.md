# SN76489AN-MicroPython
MicroPython v1.17+ script for SN76489AN hardware music playback

To import the .VGM files please use rshell  
https://github.com/dhylands/rshell  
`$ rshell`  
`> cp filepath/filename.VGM /pyboard`
